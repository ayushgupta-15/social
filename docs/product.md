# Product Scope (V1)

## Objective

Build a social web app focused on a fast, simple public posting and interaction loop.

## Must-Have Features

- Feed
- Follow / unfollow
- Create post (text + optional image)
- Like / unlike
- Comment
- Notifications (like, comment, follow)

## Explicit Non-Goals

- Chat / direct messaging
- Stories
- AI-generated content features
- Ads platform

## Users

- Authenticated users can post, follow, like, comment, and view notifications.
- Unauthenticated users can browse public feed and profiles with limited interaction.

## Success Criteria

- Users can complete the core social loop (post -> interaction -> notification) without errors.
- Feed and profile surfaces are functional on mobile and desktop.
- Data model and APIs support incremental scale from prototype to early production.
